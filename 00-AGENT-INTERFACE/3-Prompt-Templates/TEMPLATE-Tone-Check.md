# Prompt: Tone & Alignment Check

**When to Use:** Before finalizing any external communication or strategic document.

## The Prompt
```markdown
Please review the following content against the "Humble Confident Canadian" tone defined in `01-ACTIVE-BUSINESS/daily-operations/Luminous-Master-Context.md`.

**Content to Review:**
[PASTE CONTENT HERE]

**Checklist:**
1.  [ ] Does it sound "Humble Confident"? (Avoids arrogance, but asserts expertise).
2.  [ ] Does it use "Banned Words"? (AI-Native, Prediction, Graph Database).
3.  [ ] Does it use "Required Words"? (Relational Context, Correlation, Audit Trail).
4.  [ ] Does it respect the "Engineer-Safe" dictionary?

**Output:**
-   Pass/Fail status.
-   Specific rewrites for any violations.
```
