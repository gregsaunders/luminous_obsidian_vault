# Prompt: Master Context Health Check

**When to Use:** Monthly or Quarterly.

## The Prompt
```markdown
Perform a "Health Check" on `01-ACTIVE-BUSINESS/daily-operations/Luminous-Master-Context.md`.

**Tasks:**
1.  **Review Section 8 (Continuous Improvement):**
    *   Which items should be PROMOTED to the main text?
    *   Which items can be ARCHIVED?
2.  **Staleness Check:**
    *   Are there any sections that haven't been updated in >3 months?
    *   Is the "Current Focus" still accurate?
3.  **Optimization:**
    *   Can we condense any sections to keep the file length manageable?

**Output:**
A bulleted list of recommended updates to the Master Context file.
```
